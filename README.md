# IMPROTECH Alumni App

Application web de gestion du réseau alumni Improtech Training Center.

---

## 🚀 Mise en ligne sur GitHub Pages — Guide étape par étape

### Étape 1 — Créer un compte GitHub
- Va sur [github.com](https://github.com) et crée un compte gratuit si tu n'en as pas.

### Étape 2 — Créer un nouveau repository
1. Clique sur le bouton vert **"New"** (ou va sur github.com/new)
2. Nom du repository : `improtech-alumni` *(exactement ce nom)*
3. Laisse tout le reste par défaut
4. Clique **"Create repository"**

### Étape 3 — Uploader les fichiers
1. Dans ton nouveau repository, clique **"uploading an existing file"**
2. **Glisse-dépose tout le contenu** de ce dossier `improtech-alumni` dans la zone
   - ⚠️ Il faut uploader les dossiers `.github/` et `src/` aussi
   - Si GitHub ne permet pas les dossiers, utilise l'option ci-dessous avec GitHub Desktop
3. Clique **"Commit changes"**

### Alternative — GitHub Desktop (plus simple pour les dossiers)
1. Télécharge [GitHub Desktop](https://desktop.github.com/)
2. Connecte-toi avec ton compte GitHub
3. Clique "Add" → "Add Existing Repository" → sélectionne ce dossier
4. Publie le repository sur GitHub

### Étape 4 — Activer GitHub Pages
1. Dans ton repository GitHub, clique sur **Settings** (⚙️)
2. Dans le menu gauche, clique **Pages**
3. Sous "Source", sélectionne **"GitHub Actions"**
4. Sauvegarde

### Étape 5 — Attendre le déploiement
- Va dans l'onglet **Actions** de ton repository
- Tu verras un workflow "Deploy to GitHub Pages" en cours
- Après ~2 minutes, ton app sera disponible à l'adresse :
  **https://[TON-USERNAME].github.io/improtech-alumni/**
